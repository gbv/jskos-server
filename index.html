Endpoints

  • GET /status - returns a status object
  • GET /voc - returns a list of terminologies (concept schemes) (parameters: uri)
  • GET /concordances - returns a list of concordances for mappings (parameters: uri, fromScheme, toScheme, creator, mode, download)
  • GET /mappings - returns a list of mappings (parameters: identifier, from, to, mode, direction, fromScheme, toScheme, type, partOf, creator, download)
  • GET /mappings/voc - returns a list of concept schemes used in mappings (parameters: from, to, mode)
